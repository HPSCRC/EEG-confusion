Deep learning networks are garnering increasing attention across many fields. Recently, deep learning models have been applied to the identification of cognitive confusion in online education, demonstrating impressive performance. However, the scarcity of publicly available datasets has hindered progress in this area. Therefore, we have released the Cognitive Confusion EEG (CC-EEG) dataset. Our CC-EEG dataset includes 19 participants, each of whom watched 21 educational videos lasting approximately 4 minutes, resulting in a total of 399 raw EEG data samples. We hope this dataset will accelerate the development of brain-computer interfaces in the emerging field of learning confusion recognition within online education.

For more information about the dataset, please refer to the paper on the dataset:
Junxiu Liu, Jin Zeng, Senhui Qiu. Confusion recognition in online learning using EEG: A benchmark dataset and the AFN deep learning model. Biomedical Signal Processing and Control. 2026.
